Návrat Mužství is my biggest project so far — a fully functional blogging platform built using HTML, CSS, JavaScript, Node.js, Express.js, and Firebase for the database. The site features a secure registration and login system, allowing users to create accounts and manage access easily. Admins and editors can write and edit articles, insert images, and manage content through a custom editor. There's also a built-in calendar where they can add and manage upcoming events. The entire platform is fully responsive, providing a smooth and clean user experience on all devices — from phones to desktops. This project was originally developed for a client and was meant to be sold, but the client backed out after the site was completed. Even so, I'm proud of how it turned out — it's a solid example of my ability to handle large, real-world projects with both frontend and backend functionality.
